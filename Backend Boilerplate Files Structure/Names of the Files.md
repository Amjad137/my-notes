Basically in the app/ or src/ directory, there should be the following files

1. [[@types]]
2. config - the main backend config file [[app.config.js]] will be here and it will be called in index.js
3. constants
4. controllers
5. dto
6. exceptions
7. middleware
8. migrations
9. models
10. routes
11. schema
12. services -> contains 
13. utils
14. [[index.js]]
